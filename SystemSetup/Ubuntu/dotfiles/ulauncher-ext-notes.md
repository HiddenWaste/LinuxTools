# Settings
!! FIRST THINGS FIRST !!

Couple things must be done on ulauncher's settings itself
- Change theme to a dark one adwaita is preferred
- Check the box for starting at login!!
    - The super key will be disable by the install scripts so this is pretty important to not get annoyed later

# Extensions
1. ulauncher-system by iboyperson

allows for quick restarts, shut downs, etc from ulauncher!

2. spotify web api - the-lay

Much, much better than the extension that controls only desktop. this has many more features available and since it interacts with the webapi you can use it to control whatever you are listening on, not just the computer that ulauncher is running from.

--- May Require Additional Setup ---

3. Custom Scripts - NastuzziSamy

Be able to call established bash scripts from ulauncher, have used it so far for a direct ssh acces, which the script will spawn its own terminal login and everything! For work so not included in this repository sadly...

4. YouTube Search - NastuzziSamy

Another hit from Samy here, and pretty self explanatory. Allows one to search for youtube videos from ulauncher and opens a browser tab directly to them!

5. duckduckgo - mikebarkmin

Allows for ddg searching directly from the ulauncher window. This is my preferred search engine as it has an immense number of 'bangs' available for quick search on different sites, many more than I had ever imagines. I could use 'ddg fullmetal alchemist !crs' in ulauncher and it takes me to crunchyroll automatically searching fullmetal alchemist!!
